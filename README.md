# Python 101 - Data Analysis with NumPy and Pandas 🚀

Welcome to the **Data Analysis with NumPy and Pandas** project! This repository contains a comprehensive Jupyter Notebook that serves as a hands-on guide to mastering two of the most powerful and essential Python libraries for Data Science and Data Analysis: **NumPy** and **Pandas**.

## 📖 Overview

This project is structured as a series of tasks and mini-challenges designed to take you from the fundamentals of array manipulation to advanced data operations using DataFrames. Whether you are dealing with mathematical operations, data cleaning, or complex merging, this notebook covers the essential techniques you need.

## 🛠️ Key Topics Covered

### NumPy (Numerical Python)
*   **Mathematical Operations:** Performing vectorized operations efficiently.
*   **Arrays Slicing and Indexing:** Accessing specific elements, rows, and columns dynamically.
*   **Elements Selection (Conditional):** Filtering arrays based on logical conditions (e.g., extracting odd/even elements, positive/negative values).

### Pandas
*   **Pandas Fundamentals:** Introduction to Series and DataFrames.
*   **Data I/O:** Reading tabular data from CSV files and extracting data directly from HTML sources.
*   **Pandas Operations:** Filtering, adding, modifying, and deleting columns/rows.
*   **Applying Functions:** Leveraging `.apply()` to execute custom functions across DataFrame columns.
*   **Sorting and Ordering:** Organizing data efficiently based on specific criteria.
*   **Concatenating and Merging:** Combining multiple DataFrames using various join techniques.

## 🎯 Mini-Challenges & Final Project

Throughout the notebook, you will find several **Mini-Challenges** to test your understanding of the concepts introduced. 

At the end of the notebook, there is a **Final Project** that ties everything together. You will step into the shoes of a Data Analyst working for a bank:
- Managing client DataFrames.
- Concatenating new clients' data with existing ones.
- Merging salary information based on unique Client IDs.
- Adding yourself as a new client to the bank's system!

All solutions to the mini-challenges and the final project are provided at the end of the notebook.

## 🚀 Getting Started

### Prerequisites
Make sure you have the following installed:
- Python 3.x
- Jupyter Notebook or JupyterLab
- NumPy (`pip install numpy`)
- Pandas (`pip install pandas`)
- lxml for HTML parsing (`pip install lxml`)

### Running the Notebook
1. Clone this repository to your local machine.
2. Navigate to the project directory:
   ```bash
   cd Numpy_Pandas_Project
   ```
3. Launch Jupyter Notebook:
   ```bash
   jupyter notebook
   ```
4. Open `Python_101_Data_Analysis_Numpy_Pandas.ipynb` and run the cells interactively!
